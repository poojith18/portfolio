# 🚀 Poojith’s Portfolio

🌐 **Live Website:** [https://poojith18.github.io/portfolio/](https://poojith18.github.io/portfolio/)

A **modern developer portfolio** showcasing my work, projects, and skills in **Software Engineering, AI/ML, and Cloud Systems**.

---

## ✨ Features

- ⚡ Fast development and build with **Vite**
- 🎨 Sleek UI with **TailwindCSS + shadcn/ui**
- ⚛️ Built using **React + TypeScript**
- 📱 Fully responsive for mobile, tablet, and desktop
- 🧭 Smooth scrolling navigation between sections
- 🧩 Modular component architecture
- 🚀 Automatic deployment with **GitHub Pages**

---

## 🛠 Tech Stack

| Frontend       | Styling          | Deployment       | Utilities                |

Running the Project Locally

Clone the repository:

git clone https://github.com/poojith18/portfolio.git

Navigate into the project directory:

cd portfolio

Install dependencies:

npm install

Run the development server:

npm run dev

The app will start at:

http://localhost:8080

🏗 Build for Production

To create an optimized production build:

npm run build

Preview the production build locally:

npm run preview
🚀 Deployment

This portfolio is deployed using GitHub Pages.

Deployment command:

npm run deploy

This will:

Build the project

Push the production build to the gh-pages branch

Publish the site automatically
